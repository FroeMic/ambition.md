# ambition.md

A personal manifesto on building frighteningly ambitious things.

## About

This is a static website built with Next.js 16.1.4, featuring:
- **Space Grotesk font** - Optimized and self-hosted
- **Automatic dark mode** - Responds to system preferences
- **Pure black/white theme** - Minimal and focused
- **Static site generation** - No server required

## Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Static site output is generated in the `out/` directory.

## Deploy

The `out/` directory can be deployed to any static hosting service:
- Vercel
- Netlify
- GitHub Pages
- Any CDN or static host
